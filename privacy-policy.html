Privacy Policy for Enough Social Extension

TL;DR: This extension does not collect, store, or transmit any personal data. All data remains on your device.

Data Collection

This extension does not collect, store, or transmit any personal data to external servers. We respect your privacy completely.

Local Storage

The extension stores the following data locally on your device using Chrome's storage API:

  • Daily time spent on tracked websites
  • Your time limit settings
  • Reminder status and snooze settings
  • Last reset date

All data remains on your device and is never shared with anyone.

Permissions Used

This extension uses the following Chrome permissions:

  • alarms: Schedule daily resets and time tracking intervals
  • idle: Detect user activity to ensure accurate time tracking
  • storage: Store time tracking data locally on your device
  • tabs: Identify which website is currently active
  • windows: Display popup notifications when time limit is reached

Data Deletion

You can delete all stored data at any time by:

  • Clicking "Reset Today" in the extension popup
  • Uninstalling the extension
  • Clearing Chrome's extension data in browser settings

Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

Updates to This Policy

If we update this privacy policy, we will notify users through the extension or GitHub repository.
